# projeto-C02

Projeto C++ da matéria de C02 - 2025

Nome da aplicação: MeloMix

Tema: Catálogo de partituras de notas musicais de instrumentos.

Integrantes: Bruna Pereira de Oliveira, Cassiano Eller Vitoriano Fonseca, Marlon Robert Mota Rosa;

Abstract: O aplicativo MeloMix feito em C++ tem o intuito de, pelo terminal, mostrar ao usuário a nota musical escolhida de determinado instrumento, além de tocar o som da mesma para o usuário.  Ele terá uma interface de login e create account e uma interface para escolher os instrumentos e seus acordes.

Funções dos integrantes: 
Bruna Pereira de Oliveira: programar o login e registro de usuário;

Cassiano Eller Vitoriano Fonseca: fazer integração dos sistemas e garantir o funcionamento do código, e criar o sistema de saída de som do aplicativo;

Marlon Robert Mota Rosa: programar a leitura dos arquivos de partituras das notas dos instrumentos.
